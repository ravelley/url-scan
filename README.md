# URL Scan
Web service API to scan a given URL and return various artifacts from it.

## Running locally

### Requirements
* NodeJS
* NPM

### Steps
1. Install dependencies
    ```
    npm install
    ```
1. Set PORT (default: 8000)
1. Start server
    ```
    npm start
    ```
1. Start dev server
    ```
    npm run dev
    ```
1. App available at localhost:8000
1. Documentation available at localhost:8000/api/v1/doc
